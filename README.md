# ⚡ PulseVision: Real-Time Signal Analytics

A modern, web-based digital oscilloscope and signal analysis dashboard designed to visualize hardware signal inputs, noise filtering, and Fast Fourier Transforms (FFT) in real time.

### 🛠️ Tech Stack
* **Backend:** Python, NumPy (Signal Generation & DSP)
* **Frontend:** React.js, WebSockets (60 FPS Data Rendering)
* **Architecture:** Microservices linking simulated hardware telemetry to a cloud dashboard.

### 🎯 Project Goals
1. Simulate noisy hardware sensor data.
2. Stream data continuously without crashing the browser.
3. Provide an interactive, responsive waveform studio.
